# AbletonGPT

Python package scaffold for AbletonGPT.

## Overview

This repository is a starting point for a Python package named `ableton_gpt`.
It includes a package layout, CLI entrypoint, configuration support, tests, and CI workflow.

## Install

```bash
python -m pip install .
```

## Usage

```bash
python -m ableton_gpt --help
```

## Development

```bash
python -m pytest
```
